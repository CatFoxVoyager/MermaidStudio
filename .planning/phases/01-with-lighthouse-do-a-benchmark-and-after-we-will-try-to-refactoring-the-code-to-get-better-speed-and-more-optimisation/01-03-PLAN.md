  <action>
1. Read both files completely to understand all imports and how components are used.

2. In `src/components/AppLayout.tsx`, convert the following static imports to lazy imports:

   BEFORE (static):
   ```typescript
   import { AIPanel } from '@/ai/AIPanel';
   import { DiagramColorsPanel } from '@/components/modals/settings/DiagramColorsPanel';
   import { AdvancedStylePanel } from '@/components/modals/settings/AdvancedStylePanel';
   ```

   AFTER (lazy):
   ```typescript
   import { lazy, Suspense } from 'react';

   const LazyAIPanel = lazy(() => import('@/components/ai/AIPanel').then(m => ({ default: m.AIPanel })));
   const LazyDiagramColorsPanel = lazy(() => import('@/components/modals/settings/DiagramColorsPanel').then(m => ({ default: m.DiagramColorsPanel })));
   const LazyAdvancedStylePanel = lazy(() => import('@/components/modals/settings/AdvancedStylePanel').then(m => ({ default: m.AdvancedStylePanel })));
   ```

   NOTE: These components use named exports, so we need the `.then(m => ({ default: m.ComponentName }))` pattern to provide a default export for React.lazy.

3. In the JSX, wrap each lazy component with `<Suspense fallback={null}>`:
   - AIPanel: only rendered when `showAI` is true, so wrap it:
     ```tsx
     {showAI && (
       <Suspense fallback={null}>
         <LazyAIPanel ...props />
       </Suspense>
     )}
     ```
   - DiagramColorsPanel: only rendered when `showDiagramColors` is true
   - AdvancedStylePanel: only rendered when `showAdvancedStyle` is true

4. In `src/components/ModalProvider.tsx`, convert the following heavy modal imports to lazy:

   ```typescript
   const LazyExportModal = lazy(() => import('@/components/modals/diagram/ExportModal').then(m => ({ default: m.ExportModal })));
   const LazyTemplateLibrary = lazy(() => import('@/components/modals/diagram/TemplateLibrary').then(m => ({ default: m.TemplateLibrary })));
   const LazyVersionHistory = lazy(() => import('@/components/modals/diagram/VersionHistory').then(m => ({ default: m.VersionHistory })));
   ```

   Wrap each in `<Suspense fallback={null}>` -- they're only shown when their respective modal flag is true.

5. Keep lightweight modals (KeyboardShortcuts, CommandPalette, etc.) as static imports -- they're small.

6. IMPORTANT: Check each component for named exports and handle them correctly with the `.then()` pattern. If a component uses `export default`, use `lazy(() => import('./path'))` directly.

7. Run `npm test -- --run` to verify all tests pass.
